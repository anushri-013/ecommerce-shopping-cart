E-Commerce Shopping Cart System

A simple and efficient e-commerce application that demonstrates core online shopping functionalities such as product listing, cart management, order processing, and payment simulation. This project is designed for learning and implementing Object-Oriented Programming (OOP) concepts with database integration.

>>Features

User registration and login

Product browsing

Add/remove products from cart

Quantity management

Order placement

Total price calculation

SQLite database support
